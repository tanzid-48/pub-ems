# PUB EMS v6 — Employee Management System
**Pundra University of Science and Technology, Bogura, Bangladesh**

## Features
- Analytics Dashboard — 6 KPI cards + 8 charts + insights panel
- Employee Management — Faculty, Staff, Admin (CRUD)
- Department Management — 10 departments
- Attendance Tracking — Daily mark
- Leave Management — Approve/Reject workflow
- Salary & Payroll — View and edit
- Bus Schedule — 5 routes management

## Requirements
- Java JDK 17+
- JavaFX SDK 21
- MySQL Server 8.x
- Git

## Setup
1. Import database: `source schema.sql` in MySQL
2. Update password in `src/main/java/com/PUB/db/DB.java`
3. Update JavaFX path in `run.bat`
4. Double-click `run.bat`
